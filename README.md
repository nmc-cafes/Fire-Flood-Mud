# Can a physics-based fire behavior model predict burn severity and post-fire debris flow hazard?

This repository contains code and data associated with our investigation of QUIC-Fire's utility in predicting post-fire debris flow hazard.
All code lives in either the [Python](/Python) or [R](\R) folders.
Derived data files (.csv) are in the root repository folder.
Raw data files and QUIC-Fire model output summaries can be found in the associated USGS Data Release here (link coming soon).
Any raw data not included in the data release is available in the following [Google Drive](https://drive.google.com/drive/folders/1L5n4bhYrXz-JiYqQbdrwyeHOwmt9Q2wh?usp=drive_link)
